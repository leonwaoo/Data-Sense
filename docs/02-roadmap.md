## Fase 1: Definicao do produto

Status: concluida.

Entregas:

- Definir problema, publico-alvo e proposta de valor.
- Definir escopo do MVP.
- Definir stack inicial.

- Definir requisitos tecnicos do MVP.

## Fase 2: Preparacao do projeto

Status: em andamento.

Entregas:

- Criar estrutura de pastas. Status: concluido.

- Configurar frontend. Status: concluido.
- Criar dataset de exemplo. Status: concluido.
- Criar guia de execucao local. Status: concluido.

## Fase 3: Upload e perfil do CSV

Status: proxima fase.

Entregas:

- Upload de CSV.
- Validacao do arquivo.
- Leitura com Pandas.
- Pre-visualizacao dos dados.
- Perfil automatico do dataset.
- Testes das funcoes principais.

## Fase 4: Auditor de qualidade dos dados

Entregas:

- Detectar valores ausentes.
- Detectar duplicatas.
- Detectar possiveis outliers.
- Detectar tipos inconsistentes.
- Gerar score de qualidade.
- Sugerir correcoes.

## Fase 5: Chat analitico

Entregas:

- Interface de chat.
- Interpretacao de perguntas.
- Conversao de perguntas em operacoes analiticas.
- Respostas com texto, tabelas e metricas.
- Tratamento de perguntas fora do escopo.

## Fase 6: Visualizacoes

Entregas:

- Gerar graficos de barras.
- Gerar graficos de linha.
- Gerar graficos de pizza ou rosca.
- Gerar graficos de dispersao quando aplicavel.
- Exibir graficos no frontend.

## Fase 7: Polimento para portfolio

Entregas:

- Melhorar interface.
- Criar screenshots.
- Criar dados de demonstracao.
- Completar README.
- Documentar arquitetura e decisoes.

## Fase 8: Deploy

Entregas:

- Escolher plataforma de deploy.
- Configurar variaveis de ambiente.
- Publicar aplicacao.
